Tentang Program Studi

Program Studi Manajemen Bisnis berfokus pada pengembangan kemampuan manajerial, kewirausahaan, serta strategi bisnis modern berbasis digital.

Visi

Menjadi program studi unggul di bidang manajemen bisnis berbasis teknologi dan berdaya saing global.

Misi

  • Pendidikan berkualitas
  • Inovasi bisnis digital
  • Pengembangan kewirausahaan
  • Kerja sama industri

Tujuan Program Studi

  • Menghasilkan lulusan kompeten
  • Membentuk karakter profesional
  • Menciptakan inovasi bisnis

Keunggulan Program

Program ini menawarkan pembelajaran berbasis praktik, dosen berpengalaman, serta peluang besar untuk mengembangkan bisnis di era digital.
